◆ configureGridShape()

template<typename F = float>
ll::vec3ui ll::configureGridShape ( const vec3ui &  localShape,
const vec3ui &  globalShape 
)
noexcept

Calculates the grid shape provided local and global shapes.

The calculation is done as:

grid.x = ceil(global.x / local.x)
grid.y = ceil(global.y / local.y)
grid.z = ceil(global.z / local.z)

This method does not control the case of dividing by zero.

Parameters
[in]localShapeThe local shape. Its values must be greater than zero.
[in]globalShapeThe global shape.
Template Parameters
FFloating point type used for the calculations. Defaults to float.
Returns
The grid shape.
